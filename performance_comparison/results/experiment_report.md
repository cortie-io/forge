# 성능 비교 실험 보고서 (자동 생성 예시)

## 1. 실험 개요
- 120문제 샘플을 대상으로 세 가지 방식(GEMMA 단독, RAG+GEMMA, 전체 파이프라인)으로 해설 생성 및 성능 비교

## 2. 실험 방법
- 각 방식별로 동일한 문제를 입력하여 해설 생성
- 성능 평가지표(정확도, BLEU, ROUGE 등)로 수치화
- 결과를 표와 그래프로 시각화

## 3. 결과 요약
- [performance_table.csv]와 [performance_graph.png] 참고

## 4. 결론 및 해석
- 각 방식의 장단점 및 실제 적용 시 고려사항 기술
- 추가 실험/개선 방향 제안
