# forge-web

A minimal, enterprise-grade Next.js app inspired by Raycast's high-end dark UI.

## Features
- Raycast-style near-black blue background
- Inter font, positive letter-spacing, macOS-style shadows
- Minimal navigation, hero, and feature card components
- Strict TypeScript, ESLint, Prettier

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the app.
